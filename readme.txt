API integration for the WooCommerce REST API. Send data to any APIs, code free, free to use and easy to setup.

== Description ==

Easily create your own API integrations in WordPress in just a few minutes.
AnyAPI provides a user-friendly setup interface suitable for beginners to experts.

Supports authentication: Basic Auth, Bearer Token.

Tested with Wordpress 6.7 and WooCommerce version 9.4.1.

*** Features ***
 * OrderAPI integration among Wordpress, WooCommerce, and any REST API.
 * Triggers most WooCommerce order API actions.
 * Custom filter for WooCommerce order API JSON fields.
 * Easy management of API integrations through a user-friendly interface.
 * No coding required.
 * WooCommerce RESTAPI Tools - CRUD methods.


== Frequently Asked Questions ==

= Why / Who use AnyAPI ? =

AnyAPI is a simple REST API integration tool that connects WordPress and WooCommerce to any REST API.
Our WooCommerce API integration is perfect for business owners, project managers, developers,
and anyone else who wants to integrate APIs without writing code.

= What is OrderAPI integration ? =

If you want to send data to any REST API when a customer purchases a product on your website,
you can create an OrderAPI integration to trigger actions and send out the purchase data.
